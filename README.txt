WASSER.MARKET v1.1

Open index.html in a browser.
German is the default language. Use the DE/EN switch in the header.
Product search and comparison fields provide live suggestions from the first typed letters.
All product and laboratory data is demonstration content and must be verified before publication.

FAVICON
The approved WASSER.MARKET droplet icon is connected to every HTML page.
Browser, Apple Touch and Android icon files are stored in assets/favicon/.


REVIEWS AND SEO
Ratings/comments work locally via localStorage in this static prototype. Shared public reviews require a server database and moderation API. SEO files include robots.txt, sitemap.xml, JSON-LD, canonical, Open Graph, Twitter metadata, llms.txt and ai-policy.txt. Replace https://wasser.market if the production domain differs.
